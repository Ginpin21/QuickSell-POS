Requirements:
1. XAMPP V3.3.0 or Higher
2. Microsoft Visual Studio 2022 

Steps to run:
1. Run XAMPP.
2. Start Apache Web Server and MySQL.
3. Click on MySQL Admin.
4. Create a new database named pos_system.
5. Import the pos_system.sql file, disable foreign key checks.
6. Open the QuickSell POS\bin\Release\QuickSell POS.exe file.
7. Use username "Admin" and Password "123" to login to Admin Window.
8. You can add new users or login with previous user accounts by reopening the application.
